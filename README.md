# Oxy949's A/V Client VDO.Ninje (Module)
Replacement for Foundry VTT v13 built-in A/V. Shows VDO.Ninja streams inside Foundry Camera Views

## Installation
1. Copy https://github.com/Oxy949/fvtt-avclient-vdoninja/releases/latest/download/module.json
2. Paste it in your Foundry VTT, wait for install
3. Enable the module in your world
4. Make shure you enable A/V Chat in your world (if no ssh, use module settings to force enable it)
5. Create new Ninja room and join it with all the players
6. Copy player solo view url's (from director panel) and paste it in module settings
7. Enjoy your players cameras!
